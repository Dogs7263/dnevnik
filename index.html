Как считается рейтинг?

Рейтинг рассчитывается на основе среднего балла по всем предметам:

  • Учитываются все оценки по всем предметам
  • Высчитывается средний балл для каждого ученика
  • Ученики сортируются по убыванию среднего балла
  • Первые три места отмечаются специальными значками

Топ-3 ученика получают особое оформление в рейтинге!
